# Specification-Driven Development (SDD) with Speckit and Gemini CLI

This repository hosts a **tutorial** and a **practical experimentation** of the **Specification-Driven Development (SDD)** methodology.

It uses the [**Speckit**](https://github.com/github/spec-kit) and [**Gemini CLI**](https://github.com/google-gemini/gemini-cli) for integrating generative AI into the development workflow.

In this repository you'll find:

- the [tutorial itself](./sdd.md)
- the generated / provided specs and code.
